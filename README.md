Helipad Buddy is an advanced aviation sensor console designed to provide real-time environmental and flight-awareness data directly from onboard device sensors.

Features include:
• QFE / QNH pressure calculations  
• GNSS satellite monitoring  
• Vertical speed indicator  
• Track vs heading  
• Ground speed  
• Smart altitude handling  
• Offline capability  

⚠️ Note:
GPS altitude may vary depending on device hardware. Manual altitude entry is recommended for maximum pressure accuracy.
